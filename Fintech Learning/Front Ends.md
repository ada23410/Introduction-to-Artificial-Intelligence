- 前端是應用程式中使用者可直接看到與操作的部分，包含介面設計與互動邏輯。前端工程師負責將資料與功能以視覺化方式呈現，確保系統易用、美觀且反應迅速。
- 網頁前端（Web Front-End）
	- - 使用 **HTML、CSS、JavaScript** 建構網頁結構、樣式與互動。
	- 常搭配框架如 **Vue.js、React、Angular** 以提升開發效率與維護性。
	- 著重在響應式設計（Responsive Design），確保網站能適應不同裝置與螢幕大小。
- 行動前端（Mobile Front-End）
	- 針對特定系統開發：
		- **iOS** 使用 **Swift 或 Objective-C**。
		- **Android** 使用 **Kotlin 或 Java**。
		- 注重介面一致性、效能與操作體驗，確保應用在多裝置上穩定流暢。
- **核心功能（Key Functions）**
	- **資料輸入與回饋**：接收使用者操作（表單、點擊、手勢）。
	- **資料顯示與整合**：從後端取得資訊並即時更新畫面。
	- **動畫與互動**：使用過渡效果與動態元件提升體驗。
	- **使用者介面設計**：規劃畫面結構與視覺層級，提升可用性與美感。

###### What do software Engineers do?
1. 需求撰寫與設計規劃
	- 負責撰寫功能需求文件，並規劃與評估 **UX/UI 設計的可行性**。
2. 介面開發與互動邏輯
	- 開發 **使用者介面（UI）** 與 **使用者體驗流程（UX logic）**，確保操作直覺流暢。
3. 程式建構與部署
	- 使用前端框架與函式庫（如 **Vue.js、React**）建構可重用元件，進行 **Build 與 Deploy**。
4. 持續優化與協作
	- 與 **產品經理與團隊** 持續合作，改進應用功能與效能。
5. 問題修復與技術債管理
	- 修復 **生產環境錯誤（production bugs）** 並處理 **技術債（tech debt）**，確保系統穩定。
6. 跨層協作與系統整合
	- 與 **後端工程師（Backend Engineers）** 合作，串接 API、測試資料流程，確保前後端整合順暢。
